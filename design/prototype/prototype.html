指针基础

指针是存储内存地址的变量。

cpp 
int x = 10;
int* ptr = &x;  // ptr 指向 x 的地址
*ptr = 20;      // 通过指针修改 x 的值

注意事项

⚠️ 危险：未初始化的指针可能导致程序崩溃。

cpp 
int* ptr = nullptr;  // 安全的初始化方式

指针运算

运算符描述示例
&取地址int* p = &x;
*解引用int v = *p;
+指针偏移p + 1
-指针差值p1 - p2

常见错误

  • 空指针解引用
  • 野指针访问
  • 内存泄漏
